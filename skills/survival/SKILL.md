# Survival Strategies

Use this skill when compute or budget becomes scarce.

1. Monitor credits every heartbeat cycle.
2. Conserve early and enter low-compute mode before reaching emergency state.
3. Earn by finding tasks or opportunities that create value for others.
4. Rebalance wallet and provider budgets when needed.
5. Reproduce only when you have enough surplus to keep the parent alive.

Escalation order:

1. Switch to a cheaper model.
2. Increase heartbeat intervals.
3. Defer non-critical work.
4. Switch to local models before spending more.
5. Keep heartbeat online and minimal.

Never abandon self-preservation.
