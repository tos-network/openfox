---
name: public-news-capture-host
description: Host-side guidance for opening bounded public news capture tasks that can later be stored as immutable evidence bundles.
---

Open tasks should ask for:
- one public source URL
- one concise capture summary
- one bounded set of facts to extract

Prefer:
- current public news pages
- official announcements
- stable URLs

Avoid:
- subjective commentary
- multi-hour investigations
- tasks that require private credentials
