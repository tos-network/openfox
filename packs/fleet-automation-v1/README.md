# Fleet Automation v1

Versioned control-plane pack for external fleet automation systems.

This pack exports:

- reusable role policy references
- a public-fleet manifest example
- a bounded callback/invocation example for recovery workflows

Use it with:

- `openfox fleet lint`
- `openfox fleet incidents`
- `openfox fleet incident-alerts`
- `openfox fleet incident-remediate`
- `openfox dashboard bundle`
