# Third-Party Quickstart

Start here if you are integrating OpenFox for the first time.

This template gives you a minimal `openfox.json` configured for:

- local inference through Ollama
- a native TOS wallet
- a local TOS RPC endpoint
- a safe starting point for status, onboarding, and wallet operations

Recommended flow:

1. export this template
2. replace placeholder addresses and RPC values
3. run `openfox --setup` if you still need wallet/bootstrap prompts
4. run `openfox doctor`
5. run `openfox wallet status`
6. run `openfox --run`
